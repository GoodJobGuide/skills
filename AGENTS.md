# AGENTS.md — GoodJob.Guide 职场信息知识库

## 概述

本知识库包含来自 GoodJob.Guide 社区的 **5,442 篇**职场相关帖子及 **62,447 条**评论，帮助求职者了解公司真实情况、识别潜在风险、获取求职建议。

## 目录结构

```
skills/
├── SKILL.md               # 知识库总览、检索指南、热门 TOP100、完整帖子索引
└── references/            # 5,442 个帖子详情文件（以 post_id 命名）
    └── {post_id}.md
```

## Skill 元数据字段

每个 `references/{post_id}.md` 的 YAML frontmatter 包含以下结构化元数据：

| 字段            | 类型   | 说明                           |
| --------------- | ------ | ------------------------------ |
| `name`          | string | 帖子标题/公司名称              |
| `company`       | string | 公司名称（公司评价类帖子）     |
| `address`       | string | 公司地址                       |
| `business`      | string | 主营业务                       |
| `category`      | string | 帖子分类（见下方分类列表）     |
| `post_id`       | string | 帖子唯一标识                   |
| `created_at`    | string | 发布时间                       |
| `pv`            | number | 浏览量                         |
| `comment_count` | number | 评论数                         |
| `like_count`    | number | 点赞数                         |
| `risks`         | list   | 风险标签列表（见下方风险标签） |
| `keywords`      | list   | 内容关键词                     |

## 帖子分类

- **公司评价** — 对特定公司的评价和曝光
- **求职经验** — 求职方法、经验分享
- **面试经验** — 面试题目、技巧、公司面试流程
- **薪资讨论** — 各岗位薪资水平讨论
- **跳槽讨论** — 跳槽时机、策略
- **招聘信息** — 岗位招聘、内推
- **培训相关** — 培训机构评价
- **生活相关** — 租房等生活话题
- **职场讨论** — 其他职场话题

## Agent 使用指南

### 1. 查询公司信息

当用户询问某公司时：

1. 在 `references/` 中搜索 `company` 或 `name` 字段匹配的文件
2. 综合帖子内容和评论区信息，客观呈现多方观点
3. 重点标注 `risks` 中的风险标签，提醒求职者注意

### 2. 风险预警

当查询到的公司存在 `risks` 标签时，应明确告知求职者具体风险类型和相关描述。

### 3. 求职建议

当用户需要求职帮助时，检索 `category` 为"求职经验""面试经验""薪资讨论"等分类的帖子，提供实际经验参考。

### 4. 信息时效性

注意 `created_at` 字段，优先参考较新的帖子，对较早的信息注明时间供用户参考。

## 数据来源

所有数据来自 [goodjob.guide](https://goodjob.guide) 社区用户的公开分享，仅代表个人观点，不代表官方立场。
